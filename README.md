# Como jogar

- Arraste o dedo horizontalmente para mover a peça
- Arraste o dedo verticalmente para rotacionar a peça
- De um toque na tela para fixar a peça
- Ao lado esquerdo do jogo, existe um botão para mover a peça para baixo
- É possivel retirar o verlay do jogo, deixando apenas as peças visiveis, dando mais realismo

# Tecnologias utilizadas
- Swift
- SwiftUI
- Combine
- ARKit

# Como executar o jogo

- Você vai precisar de um iPhone e um mac, clone o projeto no XCode e execute no iPhone
- Ou eu posso gerar um link do TestFlight, sendo possivel baixar diretamente no iPhone
